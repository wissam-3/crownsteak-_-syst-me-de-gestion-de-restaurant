# 🍽️ CrownSteak - Système de Gestion de Restaurant

## 📌 Description du projet
**CrownSteak** est une application web de gestion de restaurant développée avec le framework Django.  
Elle permet de digitaliser et simplifier la gestion des commandes, des tables et du personnel dans un restaurant.

---

## 🚀 Fonctionnalités

### 👨‍🍳 Pour les clients :
- Choisir une table
- Consulter le menu
- Passer des commandes
- Suivre l’état des commandes en temps réel
- Donner un avis sur le service

### 👨‍💼 Pour les employés :
- Gestion des commandes (serveurs)
- Préparation des plats (chefs)
- Mise à jour du statut des commandes
- Gestion de la caisse (caissier)

---

## 🛠️ Technologies utilisées
- Python
- Django
- HTML / CSS
- JavaScript
- MySql
